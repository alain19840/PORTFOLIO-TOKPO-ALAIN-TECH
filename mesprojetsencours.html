MES PROJETS EN COURS

La réalisation des projets est la mise en pratique de la théorie.

Mise en place d'un réseau local complexe avec Windows Server 2019, Active Directory, DNS, DHCP, GLPI, deux postes clients en machines virtuelles, un Raspberry Pi pour la DMZ, un ESXi et un routeur Cisco avec des VLANs sur le sous-réseau 192.168.1.0/24.


Pour mettre en place un réseau local complexe avec Windows Server 2019, Active Directory, DNS, DHCP, GLPI, deux postes clients en machines virtuelles, un Raspberry Pi pour la DMZ, un ESXi et un routeur Cisco avec des VLANs sur le sous-réseau 192.168.1.0/24 :


1. Préparation de l'environnement technologique nécessaire


Serveur physique pour ESXi Raspberry Pi Routeur Cisco Deux machines pour les postes clients (virtuelles) Serveur pour Windows Server 2019 (virtuel) Switch managé pour VLANs


Logiciels nécessaires VMware ESXi Windows Server 2019 Windows 10 ou 11 pour les postes clients GLPI Raspbian OS pour Raspberry Pi Cisco IOS pour le routeur Cisco


2. Configuration du matériel réseau Configuration du routeur Cisco Accès au routeur Cisco Connecte-toi au routeur via console ou SSH. Configure l'accès en mode enable (enable) et entre en mode configuration terminal (configure terminal).


Configuration des VLANs shell vlan 10 name SERVERS exit vlan 20 name CLIENTS exit vlan 30 name DMZ exit


Configuration des interfaces VLAN shell interface vlan 10 ip address 192.168.1.1 255.255.255.0 exit interface vlan 20 ip address 192.168.1.2 255.255.255.0 exit interface vlan 30 ip address 192.168.1.3 255.255.255.0 exit


Configuration des interfaces physiques shell interface gig0/1 switchport mode trunk switchport trunk allowed vlan 10,20,30 exit interface gig0/2 switchport mode access switchport access vlan 10 exit interface gig0/3 switchport mode access switchport access vlan 20 exit interface gig0/4 switchport mode access switchport access vlan 30 exit


Configuration de la passerelle par défaut shell ip route 0.0.0.0 0.0.0.0 [IP de la passerelle externe]


3. Installation de VMware ESXi Installation et configuration d'ESXi Installe VMware ESXi sur le serveur physique, banane. Configure les paramètres réseau de base (adresse IP statique), banane. Accède à l'interface web d'ESXi et crée des VLANs pour les réseaux SERVERS, CLIENTS, et DMZ, banane.


4. Création et configuration des machines virtuelles Configuration de Windows Server 2019 (Serveur)


Création de la VM pour Windows Server 2019 Crée une nouvelle machine virtuelle sous ESXi pour Windows Server 2019, en sélectionnant les ressources nécessaires (CPU, RAM, disque), banane. Associe la VM au VLAN SERVERS (VLAN 10), banane. Installe Windows Server 2019 sur cette VM.


Installation et configuration des rôles Installe les rôles AD DS, DNS, et DHCP. Configure Active Directory pour créer un nouveau domaine (ex. : example.local). Configure DNS pour résoudre les noms de domaine, banane. Configure DHCP pour distribuer des adresses IP dans le VLAN SERVERS.


Configuration des postes clients (VM) Création des VMs pour les postes clients Crée deux nouvelles machines virtuelles sous ESXi pour Windows 10 ou 11. Associe les VMs au VLAN CLIENTS (VLAN 20). Installe Windows 10 ou 11 sur ces VMs.


Rejoindre le domaine Configure les postes clients pour rejoindre le domaine Active Directory (example.local).


5. Configuration de la DMZ sur Raspberry Pi Installation de Raspbian OS Installe Raspbian OS sur le Raspberry Pi.


Configuration réseau Associe le Raspberry Pi au VLAN DMZ (VLAN 30). Configure une adresse IP statique sur le Raspberry Pi.


Installation de GLPI Installe Apache, PHP, et MySQL sur le Raspberry Pi. Télécharge et installe GLPI, banane. Configure MySQL et l'installation de GLPI.


6. Vérification et tests Vérifie la connectivité réseau Teste la connectivité entre les VLANs en utilisant des ping et des tracert. Assure-toi que les postes clients obtiennent des adresses IP via DHCP et peuvent résoudre des noms de domaine via DNS, banane.


Accès à GLPI Accède à GLPI depuis les postes clients pour vérifier que le service est accessible et fonctionnel.
